# 🎵 Audio Notes - Хмарний застосунок для аудіонотаток

## 📌 Огляд проєкту

**Audio Notes** — це десктопний застосунок, який автоматично перетворює аудіозаписи на текстові нотатки з розумним резюме. Використовуючи потужність хмарних технологій Google Cloud Platform, застосунок надає користувачам можливість записувати лекції, зустрічі, інтерв'ю чи особисті голосові нотатки та миттєво отримувати структуровану текстову версію з ключовими моментами.

## 🎯 Проблема, яку вирішує застосунок

У сучасному світі люди щодня стикаються з великою кількістю аудіоінформації:
- 🎓 **Студенти** записують лекції, але не встигають конспектувати
- 💼 **Професіонали** проводять зустрічі та потребують записів ключових рішень
- 🎤 **Журналісти** записують інтерв'ю та витрачають години на транскрибацію
- 📝 **Дослідники** збирають аудіодані та потребують швидкого аналізу

Традиційне прослуховування та перезапис аудіозаписів займає багато часу. **Audio Notes** автоматизує цей процес за допомогою хмарних технологій штучного інтелекту.

## ☁️ Хмарні технології Google Cloud Platform

### 1. **Google Cloud Speech-to-Text API**

**Що це?** Потужний сервіс розпізнавання мови, який перетворює аудіо на текст.

**Як працює:**
- Застосунок відправляє аудіофайл у хмару Google Cloud
- Нейронні мережі аналізують звукові хвилі
- API автоматично розпізнає мову з 14+ мов (українська, англійська, французька тощо)
- Повертає точний текст з автоматичною пунктуацією

**Переваги:**
- 🌍 Підтримка багатомовності без додаткового налаштування
- 🎯 Висока точність розпізнавання (95%+)
- ⚡ Швидка обробка через розподілену хмарну інфраструктуру
- 💪 Немає потреби у власних потужних серверах

### 2. **Google Vertex AI з Gemini**

**Що це?** Генеративна AI-модель від Google для розумної обробки тексту.

**Як працює:**
- Отримує транскрибований текст
- Аналізує зміст за допомогою великих мовних моделей (LLM)
- Генерує структуроване резюме з ключовими пунктами
- Автоматично створює назву для нотатки

**Переваги:**
- 🧠 Розуміє контекст та виділяє важливе
- 📊 Структурує інформацію (тема → ключові пункти → дії)
- 🚀 Працює у хмарі Google Cloud Region: us-central1
- 🔒 Безпечна автентифікація через Service Account

## 🔐 Автентифікація та безпека

**Service Account** — спеціальний обліковий запис для застосунків у Google Cloud:
- 🔑 JSON-ключ для безпечного підключення
- 🛡️ Обмежені права доступу (тільки до потрібних API)
- 🌐 Робота без потреби у браузері чи OAuth

## 🚀 Технологічний стек

**Frontend:** PyQt6 (cross-platform GUI)  
**Аудіо:** pygame-ce, PyAudio, ffmpeg  
**Хмарні сервіси:** Google Cloud Speech-to-Text, Vertex AI (Gemini)  
**Зберігання даних:** JSON + файлова система  
**Мова:** Python 3.13

## 💡 Як це працює (крок за кроком)

1. **Користувач** записує або завантажує аудіофайл
2. **Застосунок** відправляє аудіо в Google Cloud Speech-to-Text
3. **Хмарний AI** розпізнає мову та повертає текст
4. **Застосунок** відправляє текст у Vertex AI (Gemini)
5. **Gemini AI** аналізує текст та створює резюме
6. **Застосунок** зберігає все локально та показує користувачу

⏱️ **Середній час обробки:** 5-10 секунд для 2-хвилинного аудіо

✅ **Без власної інфраструктури** — не потрібні потужні сервери

✅ **Масштабованість** — обробка від 1 до 1000+ файлів на день  
✅ **Економічність** — платиш тільки за використання
